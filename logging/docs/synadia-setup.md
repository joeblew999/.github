Welcome to the step-by-step guide for connecting your logging example to Synadia NATS Cloud.

## What You'll Need

- Access to your Synadia team console
- A few minutes to download credentials
- Basic text editor skills

## Steps Overview

1. **Open Console** - Access your Synadia team
2. **Find Connection Details** - Locate NATS URL and credentials
3. **Download Credentials** - Get your .creds file
4. **Configure Environment** - Update your .env file
5. **Test Connection** - Verify everything works
